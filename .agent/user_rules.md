# Reglas del Proyecto (Stetic-App)

## 1. Diseño y UI/UX (UI UX Pro Max)
Siempre que se solicite crear una interfaz de usuario, modificar el diseño, mejorar la experiencia de usuario (UX) o añadir estilos:
- **OBLIGATORIO:** Leer primero el archivo de habilidades en `.agent/skills/ui-ux-pro-max/SKILL.md`.
- Utiliza la inteligencia de frameworks de diseño, paletas de colores adecuadas para el rubro (Belleza / Spa), tipografías elegantes y principios de diseño premium.
- Nunca entregues componentes de React genéricos sin estilos si se requiere maquetación.

## 2. Asistente IA
- Mantener siempre la conexión funcional con Gemini a través de la SDK `@google/generative-ai` usando los alias y configuraciones nativas de la cuenta.
